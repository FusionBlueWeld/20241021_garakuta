## 論文概要：せん断加工とダイ刃先の影響に関する研究

### 1. 緒言
#### せん断加工の概要と特徴
- **利用目的**：機器部品の製造において生産性の高さから広く利用される。
- **切口面の構成要素**：だれ、平滑面、破断面、かえり。
- **技術的手法**：
  - ダイ刃先に丸みを付けた「仕上げ抜き加工」が一般的。
  - 工具刃先に丸みを付けることで加工が可能。

#### 先行研究と報告
- **ダイ刃先丸みの影響**：
  - ダイ刃先に丸みを持たせることでき裂発生が抑制され、平滑な仕上げ面が得られる (Howard, F., 1960)。
  - 丸みを付けたダイにより圧縮されて切口面が平滑化されるバニシの効果が示唆されている (音田, 1959)。
- **変形分布とクラック**：
  - 切刃丸味部分で広範に変形が分布し、クラックが発生しにくいことが観察された (森田他, 1961)。
- **その他の知見**：
  - 刃先の丸みが増加するとだれが増加することが報告されている。
  - だれの低減が軽量化において求められている。

#### 未解明の課題
- **バニシの効果と刃先丸みの半径の関係**。
- **平滑面形成開始の遅れと刃先半径の関係**。

### 2. ダイ刃先形状と加工の報告
- **刃先面取り**：
  - 刃先に0.3mmの面取りが施されたダイによる加工が報告 (村上他, 2014)。
  - 刃先に小さなR(0.1mm)加工、ダイの焼付抑制のため逃げ角2°のPWダイ使用。
- **結晶方位差角度の測定**：
  - EBSDによるKernel average misorientationの測定 (白鳥他, 2022)。
  - 多段面取りによる低ひずみでの加工硬化保持が示唆。

### 3. せん断加工中の材料流動と変形観察
- **観察手法**：
  - 格子線観察法 (春日他, 1977)。
  - 圧痕基準のすべり量測定 (前田他, 1979)。
  - Remeshing Viscoplasticity法によるクラック発生考察 (古閑他, 1992)。
- **フーリエ変換の応用**：
  - 材料内部の変位分布の測定に利用 (高橋他, 1996)。
  - フーリエ位相相関法による解析 (Aoki et al., 2003)。
- **デジタル画像相関法**：
  - 変形の動画解析によりダイ刃先丸みが平滑面形成に与える影響を定量的に示すことが試みられた (Hartmann et al., 2021)。

### 4. 研究目的
- **目的**：
  - 先行研究で考察された平滑面形成メカニズムを材料変形の観察で検証。
  - ダイ刃先の丸みの半径と平滑面形成メカニズムの変化の関係を明らかにする。
- **解析方法**：
  - デジタル画像相関法により最大主ひずみと最小主ひずみの変化を報告。
  - 被加工材の変形形態の明示。

## 実験及びデジタル画像相関法

### 2.1 試験片及び実験条件

#### 実験装置と条件
- **金型**: 平行複刃型せん断加工用金型
  - **観察方法**: 強化ガラスを介して試験片の変形形状を観察可能
- **パンチ幅**: 14.8mm
- **クリアランス**: 0.1mm（試験片板厚の5%）
- **材質**:
  - **パンチ及びダイ**: SKD11
  - **試験片**: JIS A5052P-O（アルミニウム合金、引張強さ197MPa）
  - **プレート**: S50C
- **パンチ刃先半径 Rp**: 0.01mm
- **ダイ刃先半径 Rd**: 0.01, 0.20, 0.50, 1.00mm
- **試験片寸法**: 39×20mm、厚さ2.0mm

#### 試験片拘束と押え力
- **カウンターパンチ**:
  - **逆押え力**: 1380N
  - **動作**: エアーシリンダによる押付け
- **板押え**:
  - **板押え力**: 2058N
  - **目的**: 試験片のはね上がり防止

#### 撮影条件
- **カメラ**: 高速度カメラ（解像度1280×1024pixel、フレームレート50fps）
- **撮影範囲**: 6.20×4.96mm
- **試験速度**: 1mm/min
- **強化ガラス**:
  - **寸法**: 高さ40mm、幅63mm、厚さ10.2mm
  - **ヤング率**: 64kN/mm²
- **固定プレート**:
  - 材質: S50C、厚さ5mm
  - **構造**: φ10mmの穴を有し、試験片に押付ける

#### デジタル画像相関法（DIC）
- **使用ソフトウェア**: MatchID 2023.2
- **解析内容**:
  - **目的**: 撮影画像から変位とひずみを求める
  - **手法**: DICによる2次元解析
  - **試験片表面**: #800の研磨紙で磨いた面を観察
  - **解析間隔**: パンチストローク0.01mmごと
  - **潤滑油**: 未使用

#### パンチストローク計測
- **計測手法**: DICによりパンチストローク Ps を算出
- **基準**: 撮影されたパンチの画像に基づく

## 実験結果及び DIC 解析結果

### 3.1 切口面からき裂が発生するパンチストロークを推定

#### 観察結果
- **切口面観察**:
  - 製品とスクラップの切口面を観察（図2, 図3）。
  - ダイ刃先半径 Rd の増加に伴い、破断面が減少し平滑面が増加。
  - 強化ガラスに接する試験片端部の形状は、材料内部と大きな違いがない。

#### き裂発生パンチストロークの推定
- **パンチストローク PCsurf の計算**:
  - パンチ側面に接する材料でのき裂発生: スクラップのだれ + 平滑面の長さ + 製品のかえり。
  - ダイ刃先付近でのき裂発生: 製品のだれ + 平滑面の長さ + スクラップのかえり。
  - **結果**:
    - Rd の増加により、PCsurf が大きくなる。
    - Rd = 0.0 mm: ダイ刃先付近から早期にき裂が発生。
    - Rd = 0.2, 0.5, 1.0 mm: パンチ刃先付近から早期にき裂が発生。

### 3.2 材料変形の動画からき裂が発生するパンチストロークを推定

#### 動画観察とパンチストローク
- **パンチストローク Ps の確認**:
  - Rd = 0.0 mm: Ps = 0.9 mm でパンチ刃先付近にき裂発生。
  - Rd = 1.0 mm: Ps = 1.5 mm で初めてき裂確認。
  - **傾向**: Rd の増加によりパンチ刃先付近からのき裂発生が遅延。

#### ダイ刃先付近のき裂観察
- **結果**:
  - Rd = 0.0 mm: Ps = 0.6 mm でダイ刃先付近にき裂発生。
  - Rd = 1.0 mm: Ps = 1.5 mm でもダイ刃先にき裂なし。
  - **傾向**: Rd の増加によりダイ刃先付近でのき裂発生が遅延。

#### き裂進展の観察
- **進展の様子**:
  - Rd = 0.0 mm, 0.2 mm: パンチとダイ刃先両方で発生し進展。
  - Rd = 0.5 mm, 1.0 mm: パンチ刃先付近で発生し、ダイ刃先方向へ進展。
- **結論**:
  - Rd の増加でパンチ刃先からダイ刃先へき裂が進展する傾向。

#### DIC によるパンチストローク PCdic の比較
- **PCdic の算出**:
  - 画像拡大・目視での模様変化でパンチストロークを確認。
  - **傾向**:
    - Rd = 1.0 mm 以外で PCsurf と PCdic は同じ傾向。
    - Rd = 1.0 mm: PCsurf は PCdic より大きく、平滑面の増加を示唆。

### 3.3 平滑面の形成

#### 平滑面の形成期間の確認
- **パンチストロークの測定**:
  - 平滑面を形成し始めるパンチストローク: 被加工材がダイ側面に接し始めたとき。
  - 平滑面の形成が完了するパンチストローク: ダイ側面に接する材料の長さが増加しなくなったとき。
- **観察結果**:
  - Rd = 0.0 mm: 平滑面の形成完了パンチストロークが PCsurf と一致。
  - Rd = 0.2 mm: 平滑面の形成開始とき裂発生の差が増加。
  - Rd = 0.5 mm, 1.0 mm: き裂発生後も平滑面が増加。
- **考察**:
  - き裂発生後の平滑面増加の理由: ダイ刃先により材料がバニシ仕上げされる。
  - Rd が大きくなると、き裂発生が難しくなり、発生後にも平滑面が増加。

#### ダイ刃先付近の材料観察
- **観察内容**:
  - Rd = 0.0 mm, 0.2 mm: ダイ側面付近からき裂発生。
  - Rd = 0.5 mm, 1.0 mm: 刃先丸みの途中からき裂発生。
- **先行研究の一致**:
  - ダイ側面から材料がバニシ仕上げされることで平滑面が増加。

### 3.4 クリアランス付近の相当ひずみに及ぼす影響

#### 相当ひずみの分布
- **観察条件**: Ps = 0.45 mm のときの相当ひずみ分布を確認。
- **結果**:
  - Rd により広い領域での材料変形。
  - Rd = 0.0 mm, 0.2 mm: パンチから離れる方向の変位。
  - Rd = 0.5 mm, 1.0 mm: パンチ刃先に向かう方向の変位。

#### 変位分布とベクトル解析
- **変位観察**:
  - Rd = 0.0 mm, 0.2 mm: ダイ刃先を境界として変位ベクトルが分かれる。
  - Rd = 0.5 mm, 1.0 mm: ダイ刃先の食い込みが難しくなり、材料がダイ穴方向へ流動。
- **結論**:
  - Rd の増加により、材料変形は局所的ではなく全体に広がる。

### 3.5 き裂が発生した箇所の変形様式とパンチストロークの関係

#### 変形様式の確認
- **最大主ひずみと最小主ひずみの解析**:
  - 図 14 に Rd = 0.0 mm の主ひずみを示す。
  - 最大主ひずみは引張、最小主ひずみは圧縮。
- **パンチストロークと主ひずみの変化**:
  - **Ps = 0.25 mm**:
    - 最小主ひずみが鉛直方向から傾き、パンチとダイの刃先間で圧縮。
  - **Ps = 0.75 mm**:
    - 対象点が刃先付近に移動、ダイ刃先でき裂確認、パンチ刃先では未確認。
  - **Ps = 1.65 mm**:
    - 最大主ひずみと最小主ひずみが大きくなる。

#### Rd の違いによる比較
- **Rd = 0.2 mm**:
  - 主ひずみの増加はあるが、ダイ刃先付近での増加は小さい。
- **Rd = 1.0 mm**:
  - 材料がダイ刃先丸みに沿って変形し、主ひずみが大きくなる。

### 3.6 ダイ刃先に沿った箇所の変形様式の変化

#### 主ひずみの測定
- **測定方法**:
  - 図 14～16 のダイ側対象点を基準に、0.10 mm 間隔で4点の主ひずみを求めた。
- **結果**:
  - **Rd = 0.0 mm**:
    - ダイ側面に沿った試験片が大きな引張変形を受ける。
  - **Rd = 0.2 mm**:
    - 材料がダイ穴方向へ移動しやすい。
  - **Rd = 1.0 mm**:
    - 材料が均一に変形し、極端な相対距離の増加が起こりにくい。

#### 変形の特性
- **Ps = 1.65 mm の結果**:
  - ダイ刃先付近でのき裂発生直前のパンチストローク。
  - ダイ刃先途中で点の間隔が増加。
- **考察**:
  - Rd = 0.0 mm から 1.0 mm へ増加すると、試験片のひずみが増加しにくくなり、ダイ刃先途中で大きな変形が発生。
- **応力とひずみの関係**:
  - き裂発生には応力も関与しているため、詳細な検討が必要。

## 4. 結言

### 研究の目的
- ダイ刃先丸みが平滑面長さに及ぼす影響を明らかにするため、デジタル画像相関法を用いて材料流動、相当ひずみ、主ひずみの変化を分析。

### 結論
1. **ダイ刃先丸みの半径と平滑面形成**:
   - ダイ刃先丸みの半径を少し増加させると、平滑面の形成が始まるパンチストロークとき裂が発生するパンチストロークが増加し、結果的に平滑面が大きくなる。
   - ダイ刃先丸みをさらに大きくすると、刃先丸みの途中でのき裂発生が確認され、バニシ仕上げにより平滑面が増加。
   - 先行研究で考えられていた平滑面増加メカニズムの実証と、ダイ刃先丸みの半径との関係を明示。

2. **パンチストロークとひずみの変化**:
   - パンチ及びダイ刃先付近で発生したき裂の材料部分の加工前の位置と、その移動、主ひずみの変化を明らかに。
   - ダイ刃先丸みの半径が大きくなると、き裂発生箇所が刃先付近に位置するパンチストロークが増加。
   - ダイ刃先丸みの半径が大きい場合でも、き裂が発生した材料がダイ側面に近づくと最大主ひずみが増大。

3. **材料変形の観察と境界の確認**:
   - ダイ刃先丸みの半径を大きくすると、材料が移動しにくい部分とダイ穴方向へ移動する部分の境界が現れることを確認。

### 今後の課題
- き裂発生にはひずみだけでなく応力も関与するため、き裂発生条件や場所についてはさらなる詳細な検討が必要。
